# 🛍️ Snitch

A modern fashion e-commerce platform inspired by premium clothing brands, built using the **MERN Stack**.

## ✨ Features

* 🔐 JWT Authentication & Authorization
* 👤 User & Seller Roles
* 🛍️ Product Management
* 🎨 Product Variants (Color & Size)
* 📸 Image Uploads with ImageKit
* 🛒 Shopping Cart
* 📱 Fully Responsive Design
* ⚡ RESTful API Architecture
* 🔒 Secure Password Hashing (bcrypt)

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/your-username/snitch.git
cd snitch
```

### Install Dependencies

```bash
# Frontend
cd client
npm install

# Backend
cd ../server
npm install
```

### Start Development Server

```bash
npm run dev
```
